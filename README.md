# Digital Backend Controller for a mixed signal IC chip
Course Project of EE619, IIT Kanpur 

The task was to design a backend for a mixed signal IC. The chip consists of two amplifiers and an oscillator. The backend has to ensure that these analog modules are initialized correctly and the chip is ready for use. The backend has to execute a start-up sequence for this.

The entire code was organized in the following files:
• main.v - This contains the main module which instantiates other modules
and does the appropriate wiring.
• serial.v - This contains the module that is responsible for the serial com-
munication between the FPGA and the backend chip.
• startup.v - This contains the module that generates appropriate resets for
the startup sequence. A Moore type finite state machine (FSM) was used
for this task.
• tb.v - This contains a comprehensive testbench that was generated by
Claude (AI assistant) to check the functionality of the code.
